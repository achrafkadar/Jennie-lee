La nouvelle landing n’est pas encore en ligne sur Vercel

L’adresse jennie-lee.vercel.app n’existe pas tant qu’aucun projet Vercel n’a été créé et déployé. D’où l’erreur 404 NOT_FOUND.

À faire : déployer le dossier web (Next.js) sur Vercel. Ensuite, remplacez l’URL dans ce fichier par votre vraie URL (recherche VERCEL_APP_URL ci-dessous).
  1. Ouvrir Vercel et importer le dépôt GitHub Jennie-lee.
  2. Dans Project Settings → General → Root Directory, indiquer web (obligatoire : l’app Next n’est pas à la racine du repo).
  3. Ajouter les variables d’environnement (voir web/.env.example), puis Redeploy.
  4. Copier l’URL du site (ex. https://jennie-lee-xxxxx.vercel.app) et la coller à la place de VERCEL_APP_URL dans ce fichier pour activer une redirection depuis GitHub Pages (optionnel).
